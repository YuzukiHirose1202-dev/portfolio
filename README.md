# WearLog

## 概要
服装を記録・管理できるアプリです。

## 背景
毎年、「去年のこの時季は何を着ていたっけ」が起こるため、写真を入れるだけで手軽に服装を記録できるものを作りたいと思った。

## デモ
![デモ](./images/WearLog.gif)

## 使用技術
- React
- JavaScript
- OpenWeather API

## 機能
- コーディネート記録
- 画像保存
- 画像プレビュー表示
- 日付および天気情報の自動入力

## 工夫した点
- UIをシンプルに設計し、一目で内容がわかるようなレイアウトにした
- ユーザーが入力の手間を減らせるように、日付と天気を自動入力にした
- 画像選択時にプレビューを表示することで、視覚的に確認しやすくした

## 環境変数
OpenWeather APIキーを使用しています。実行する際は.envファイルを作成し、以下を設定して下さい。

REACT_APP_API_KEY=あなたのAPIキー
